Software Release 22
Monday, 17 July 2006
Software Release 22 is now available, including more fixes to the menu scripts and configuration.  These fixes resolve some issues with PS1 video fixes.
Software Release 21
Monday, 17 July 2006
Software Release 21 is now available.  This release incorporates some bug fixes as reported in the Crystal Chips forums as well as a few bugs that we found ourselves.  This release may or may not fix the problem with CC 1.2(CC1.1 with 128kbyte flash).  If not, we'll make another release as soon as we can.
Software Release 20
Saturday, 15 July 2006
We're proud to announce Crystal Chips Software Release 20, introducing BootManager 2.0 Beta!  This software includes many features such as "themes"(aka "skins"), fully-scripted menus, translatable text and more.  Get it now in the Firmware section.  Please note, however, that this is a beta release and has some known bugs.  We intend to address these issues in the coming days.  Enjoy!
Release Delayed
Tuesday, 11 July 2006
Since our announcement 2 weeks ago we've been working hard to finish up our next software update.  Many features have been finalized and bugs fixed, however there're a few remaining bugs which cause enough problems that we don't feel BM 2.0 is ready for release yet.  We're going to push the release of BM 2.0 back to Friday July 14th 2006 to allow us more time to fix these issues.  We apologize for the inconvenience this may cause but we believe it makes more sense to delay than to release software with known issues.
