Release 28
Wednesday, 06 September 2006
We've uploaded Release 28 of the Crystal Chips software.  This release fixes a problem where Auto-start of HDD drivers would cause BM to crash on some systems.
Release 27
Wednesday, 06 September 2006
We've uploaded Release 27 to resolve an issue with NTSC TVs on the installer disc.  cleanit.bat was not cleaning out the configuration files resulting in PAL configuration settings used for all consoles.  Additionally a debug option has been removed from the main menu.  
Release 26

Due to a typo in the firmware of release 25 we've uploaded release 26.  This typo caused the firmware to attempt to boot BootManager from Memory Card Slot 2 instead of Slot 1 for CC1.0-CC1.2 chips.
Crystal Chip 2.0 Pro & BootManager 2.0.6 released
Tuesday, 05 September 2006

We're happy to announce the release of Crystal Chips BootManager 2.0.6 and the availability of the new Crystal Chips v2.0! 

The CC 2.0 has 1 megabyte of DataFlash on-board which can be used like any other device with BootManager and other applications.This means you can install drivers, themes and homebrew applications directly to your Crystal Chip without wasting space on your memory card or having to have a certain card installed to run BM.  What's more, the new BootManager 2.0.6 now has PS2 HDD, USB HDD and Network support built-in allowing you to install, remove and run homebrew applications to and from any of these devices.  In addition we've fixed many of the remaining bugs, added 480P support for BM, updated many parts of the menus and added some great scripts like the TV Wizard to help you automatically configure BM for your TV! 

There's so much to tell that we've created a new BootManager Feature List to cover all the features of BM 2.0.6.  All Crystal Chips owners are invited to download the latest CC Software Release 25 from our Downloads section. For a full list of changes and fixes please view the readme.
 
One important note is that PS2 HDD support requires you to create a partition on your PS2 HDD using Unofficial Launch Elf or other HDD Management software. This partition should be named "Crystal". Enjoy and look for more exciting new features coming to the Crystal Chip devices in the following weeks! We have a new feature in development we know you're going to like.

Many thanks to all users and partners, especially to our friends at www.neoasic.com, www.foundmy.com and www.norcalmods.com for beta testing and providing installation diagrams. You have been a great help!
