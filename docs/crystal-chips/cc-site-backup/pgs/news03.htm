Software Release 32
Friday, 02 February 2007
We're happy to announce that we've uploaded CC Software Release 32!  This release includes a number of fixes, improvements and general optimizations.  Among the changes are resolution to the "black screen" when trying to boot Release 31 upgrade disc, PAL PS2 video fixes and much more.  See the readme.txt for more information.

On a side note, we've had reports of Release 30/31 not working with CC1.1 chips.  If this issue has not been resolved with Release 32, we will be putting out another release as soon as we can confirm and fix the problem.

Thanks again to all of our dedicated customers, we hope you continue to enjoy your Crystal Chip!

Reminder, Recovery mode
Wednesday, 17 January 2007
This is just a reminder to everyone upgrading to the latest BootManager software.  You MUST boot the upgrade disc in Recovery Mode, or Minimal Mode or you WILL get
 a black screen or crash at the CrystalChips logo when booting the upgrade disc.  For information on using Recovery Mode see "Hardware Modes" in the "Documents"
 section at the Official Crystal Chips website, http://www.crystal-chips.com/.
 
 We're also in the process of investigating some issues reported with the latest release.  Please be patient while we track down the cause(s).
Software Release 31
Friday, 12 January 2007
We've just uploaded CrystalChips Software Release 31 to resolve a problem with loading PS1 discs from BM.  This was caused by a minor path error in BOOTMENU.PBT.

Also, we'd like to let people know that if they've got BootManager currently installed on their PS2 that they should use Recovery Mode to boot the upgrade disc.  Some of the older BootManager software can interfere with the latest version.
