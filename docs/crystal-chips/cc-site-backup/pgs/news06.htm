BootManager 2.0.6 - short delay
Monday, 04 September 2006
Unfortunately we must delay the release of BootManager 2.0.6 while we finish debugging some of the many new features added since the previous release.  We don't expect this delay to be more than 1 or 2 days.  Thank you for your understanding, we promise you'll find it worth the wait. :)
CC compatible with PS2 v15 / CC2.0 on the way
Tuesday, 29 August 2006
We're happy to announce that existing Crystal Chips devices are fully compatible with the latest "v15"  PS2 SCPH-7700x.  Installation diagrams will be available within the next few days, thanks to the www.neoasic.com team. 

After a few weeks of public and private beta testing and further development we're nearing the next release of our BootManager software for Crystal Chips devices.  Many new features have been added and a lot of bugs fixed. Compatiblity with many applications and games has been greatly improved.  Crystal Chip 2.0 hardware is also now fully supported. We expect to release this version at the end of this week/early next week. 
 
Additionally, the new Crystal Chip 2.0 hardware will be available for sale by the end of the week. Finally, we'd like to mention that TheAgent has created a Crystal Chip "Wiki" here.
