This page is dedicated to updates and usage of the Megapack for Crystal Chips BootManager.

[edit]

What is the Mega Pack?

The BootManager MegaPack is a collection of the latest games, emulators, demos and other homebrew applications. Each application has BootManager PBAT scripts which automate the installation, loading and removal of the application so you can enjoy homebrew software without the hassle. MegaPacks may be used to install, run and remove software to/from optical disc(CD-R/RW, DVD+/-R/RW), Memory Card, USB HDD, PS2 HDD, Network, CrystalChip Flash, etc. to any of the devices you may have.

[edit]

Where can i get the Mega Pack?

The MegaPack was originally released on TheAgent's website and on the PS2-scene forums. Now the latest Mega Pack is hosted right here on the CC Wiki.

Be sure to use the version of MegaPack designed for the version of BootManager installed on your system. If you do not have the latest version of BM, get it from the CrystalChip Firmware Downloads. or try one of the CrystalChip Support Forums for customized firmwares and BootManager software.


Download MegaPack for CC BootManager v2.0.x or MegaPack for CC BootManager v2.1.x

[edit]

How do I install the Mega Pack?

Once you've downloaded the MegaPack zip file you have a few choices.


If you wish to burn a the MegaPack to CD:

1. Extract the contents of the zip file using your favorite unzip utility(winzip, winrar, etc).

2. Copy the contents of the "fw" folder to the "fw" folder of your CrystalChips Firmware.

3. Follow the usual steps to generating a CrystalChips Upgrade Disc.


If you wish to use USB HDD/Pendrive:

NOTE: Some USB drives are not supported. If your drive doesn't work, try another.

1. Extract the contents of the zip file using your favorite unzip utility(winzip, winrar, etc).

2. Copy the the entire "BM" folder the root of your drive.


If you wish to use "PS2HOST"(Network):

View the tutorial.


Once you've put the files on the media you want, check out the Application Installation Tutorial for further instructions.
