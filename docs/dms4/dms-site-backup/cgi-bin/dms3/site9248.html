Procedures on how to install DMS Explorer to memory card

Installing the DMS Explorer is rather easy and can be accomplished in just a few steps. First be sure you have a memory card with at least 560kb worth of space, Blank cd and of course, a cd burner ;-)

Download the DMS Explorer available at the downloads area. Choose whichever archive type you wish and decompress with your favorite un/archive utility. You should be left with two files bin and cue. Use the cd burning application of your choice to burn the disc image. If your burning app does not accept cue sheets then make sure you burn the bin image in mode2 xa. This tutorial will not cover how to burn the Explorer due to so many various cd burning applications on the market.

Upon burning the cd containing the explorer files, boot up your ps2 and load the disc using default (PS2) mode. If the cd was burned sucessfully then you should see this menu shown below on your screen.


The disc will automatically begin to install DMS Explorer onto your memory card. Remember, be patient as it does take a few seconds to install the explorer. Reading from memory card is inherently slow so remember this for installing and running other applications. Once complete you should see the screen shown below.


The screen will immediately give you the option of booting back to the original ps2 browser. If you have recieved this screen then all is well and you have installed the Explorer sucessfully! To return to the explorer and into Dev.olution mode simply boot your ps2 while holding the triangle button while having the memory card that contains the explorer inserted in your ps2 unit.

If for some freak chance you receive a screen such as the one shown below then you either do not have a properly formatted memory card inserted, no memory card inserted at all or not enough extra space in your memory card. Remember, the explorer requires approximately 560kb to install.
