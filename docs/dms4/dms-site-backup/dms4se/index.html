DMS4 has been at the forefront of the modding scene since it's inception. Development allows evolution and advances to make improvements and refinements. The Special Editions (S.E.) of the DMS4 offer new advantages and refinements to two very popular products. DMS Tech. have continued devlopment to make several main improvements to the DMS4 range, the new S.E. versions of Lite and Pro offer:

  • Optimized PCB layout
  • Optimized PCB size
  • Actel chip now has smaller package size
  • Hardware code revised to over increased stability and patching performance
  • Revised the installation point of Q, which now becomes C on the V12-V13 and V14 series of PS2 consoles. (Installation on V1-V11 remains unchanged)
  • Serial flash technology
  • Improved error checking
  • Improved filtering of noise

Details of new Hardware code:

 We have vastly enhanced the DMS4 SE Code to provide better patching performance, this involved editing our CDVD patching routines. We have also improved on the flash interface code by altering the communication method between the Actel and the flash. Ground bounce is not an issue on the S.E., we have fine tuned the PCB layout and optimized the FPGA routing to ensure optimal performance. The noise filtering and data error checking code of the original DMS4 has been strengthened, building on the success of the original code.

 Upgrading the S.E.:

 The new flash type significantly reduces the chances of data corruption during reflash. We also include a routine to check the integirty of data as it's programmed to the flash. Flashing is now faster and less prone to error.
The S.E. can be upgraded via CD, Network, USB or memory card.

Software:

DMS4 S.E. has some of the best Homebrew development software code onboard. Furthermore, it's stable and function packed.

Legendary Actel FPGA:

The most secure and high tech FPGA on the market today provides DMS with many developmental possibilities. We can test and improve code rapidly, We have SRAM on board and at our disposal, furthermore the Gate packed device allows us to add more Hardware features for the user to enjoy! Actel devices are secure, which means that if you buy a DMS4, chances are you're really getting a DMS4, not a cheap clone.
